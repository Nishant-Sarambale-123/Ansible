## ğŸ”¹ What Is Jinja2?

**Jinja2** is a Python-based templating engine used by Ansible to:

* Interpolate variables: `{{ variable }}`
* Apply filters: `{{ var | upper }}`
* Add logic: `{% if %}`, `{% for %}`, etc.
